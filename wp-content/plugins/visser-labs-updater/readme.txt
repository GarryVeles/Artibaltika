== Description ==

Filter the Payment Gateways by user role and cart total in your WP e-Commerce store.

For more information visit: http://www.visser.com.au/plugins/filter-payment-gateways/

== Installation ==

1. Upload the folder 'visser-labs-updater' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

Just sit back and relax - Visser Labs Updater will automatically check for updates and notify you when they are available.
